Solar-Powered EV Charging Station with Battery Energy Storage System

Overview

This project presents the design and simulation of a Solar-Powered EV Charging Station using MATLAB/Simulink. The system integrates a photovoltaic (PV) array, Maximum Power Point Tracking (MPPT), a Battery Energy Storage System (BESS), a bidirectional DC-DC converter, and a buck converter for efficient and reliable EV charging.

The objective is to maximize solar power utilization while maintaining a stable DC bus voltage and ensuring safe charging of the EV battery under different solar irradiance conditions.

Features

- P&O Maximum Power Point Tracking (MPPT)
- Boost Converter for DC Bus Regulation
- Battery Energy Storage System (BESS)
- Bidirectional DC-DC Converter
- Constant Current–Constant Voltage (CC-CV) EV Charging
- State of Charge (SOC) Protection
- MATLAB/Simulink Implementation

Software Used

- MATLAB
- Simulink

Simulation Conditions

The system was tested under the following solar irradiance levels:

- 1000 W/m²
- 800 W/m²
- 500 W/m²
- 200 W/m²
